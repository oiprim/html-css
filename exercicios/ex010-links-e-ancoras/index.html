Usando links externos

Usa a tag a para indicar o link

Usa após o link as tags target="_blank" para indicar abrir em nova guia e rel="external" para indicar ao navegador que trata-se de um link externo

Para ficar na mesma página não precisa usar target, mas pode usar target="_self"

Você pode acessar meu repositório público do GitHub

Você também pode acessar meu canal gratuíto no YouTube
